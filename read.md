🎨 2. DESIGN SYSTEM (This is where most fail)
🧊 Overall Style

Modern Minimal + Technical Elegance

Think:

Dark mode first
Glassmorphism (light use)
Soft gradients
Clean spacing
🧱 Layout Principles
1. Grid-based system
12-column layout
Consistent spacing scale
2. Visual hierarchy
Big headings
Minimal text blocks
Strong contrast
3. Section rhythm
Alternate:
Text left / Image right
Image left / Text right
🎨 Color System
Base: Dark (#0B0F19)
Accent:
Electric blue / purple gradient
Neutral greys

👉 Avoid too many colors

🔤 Typography
Headings: Bold, modern (e.g. geometric)
Body: Clean, readable

Style:

Large hero text
Tight letter spacing for headings
🧩 Components

Reusable components:

Project cards
Tech badges
Timeline blocks
Metrics counters
⚡ Animations (VERY IMPORTANT)

Use:

Framer Motion

Effects:

Smooth page transitions
Hover micro-interactions
Scroll reveal
Subtle parallax

👉 Avoid overdoing it — keep it premium
